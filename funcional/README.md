# Funcional

## Objetivo

Organizar la visión funcional de Flipa para que el equipo pueda entender cómo debe operar el producto, qué procesos cubre y qué comportamiento debe soportar el sistema.

## Alcance

Esta carpeta describe los aspectos funcionales del producto: módulos, flujos principales, casos de uso, historias de usuario y requisitos. Sirve como puente entre la lógica de negocio y las decisiones técnicas.

Equipo de producto, negocio y QA

## Contexto funcional

El sistema actual es una aplicación web responsive orientada a un rol administrador. El foco funcional inicial es soportar la evaluación y gestión de solicitudes de crédito empresarial para clientes de D1, con la posibilidad de ampliar el alcance a otros perfiles en futuras versiones.
